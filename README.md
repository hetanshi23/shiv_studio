# Shiv Studio Photography Portfolio

A modern, elegant, and premium photography portfolio website for Shiv Studio, built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Modern UI/UX:** Romantic, luxurious design tailored for wedding photography.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Portfolio Gallery:** Masonry layout with category filtering and interactive lightbox.
- **Animations:** Smooth section reveals and interactive elements using Framer Motion.
- **Service Showcases:** Detailed cards for various photography services.
- **Featured Stories:** Narrative-driven project highlights.
- **Booking Process:** Visual timeline of the client journey.
- **Contact Form:** Integrated form with validation.
- **Floating WhatsApp:** Quick contact button for instant communication.

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Gallery Viewer:** react-simple-image-viewer

## Folder Structure

```text
/src
├── components      # Reusable UI sections (Hero, About, Portfolio, etc.)
├── data            # Content data (Services, Testimonials, Stories)
├── styles          # Global CSS and Tailwind directives
├── assets          # Images, icons, and logos
├── App.jsx         # Main application assembly
└── main.jsx        # Entry point
```

## Installation

1. **Clone the repository or download the source code.**
2. **Navigate to the project directory:**
   ```bash
   cd shiv-studio
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```

## Development

To run the project in development mode:
```bash
npm run dev
```

## Production

To build the project for production:
```bash
npm run build
```

The build output will be in the `dist` directory.

## Deployment to Vercel

1. **Push your code to a GitHub repository.**
2. **Login to Vercel** (https://vercel.com).
3. **Import your project** from GitHub.
4. **Vercel will automatically detect the Vite configuration.**
5. **Click "Deploy"**.

Alternatively, you can use the Vercel CLI:
```bash
npm i -g vercel
vercel
```
